# woo_gift_physic_card

Плагин WordPress для привязки физических подарочных карт к заказам WooCommerce (номинал → номер карты из пула). Работает вместе с PW WooCommerce Gift Cards Pro.

## Требования

- WordPress 5.x+
- WooCommerce
- PW WooCommerce Gift Cards Pro (Pimwick)

## Установка

Скопировать папку плагина в `wp-content/plugins/woo_gift_physic_card` и активировать в админке.
